バブルソート

空間計算量$ O\left(N\right) $

時間計算量$ O\left(N^2\right) $

比較の回数は$ (N-1)+(N-2)+\dots+1 = \dfrac{N(N-1)}{2} $

上から順番に比較して入れ替えていくと、一番下には一番小さい値が残る。 あとはそれを要素の回数分行うだけ。

最悪の場合は挿入ソートと同じ感じかな？

a = [1, 5, 2, 9, 4, 0]

n = len(a)

for i in range(0, n-1):
  for j in range(n-1, i,-1):
    print(i, j, a)
    if a[j] < a[j-1]:
      a[j], a[j-1] = a[j-1], a[j]

print(a)
