# 相関係数

- $\displaystyle r_{xy} = \frac{\sum_{i=1}^N (x_i - \bar x)(y_i - \bar y)}{N} \bigg / \sqrt{\frac{\sum_{i=1}^N (x_i - \bar x)^2}{N}} \sqrt{\frac{\sum_{i=1}^N (y_i - \bar y)^2}{N}}$
  - $\bar x$ はxの平均, $\bar y$ も同様
  - $N$ はサンプルサイズ(データ数)
- $r_{xy} = \dfrac{C_{xy}}{S_x S_y}$
  - $C_{xy}$ は共分散(covariance)
