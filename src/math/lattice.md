# 束

- 以下の公理的な恒等式を満たす、2種類の2項演算と集合
  - 可換律
    - $a \lor b = b \lor a$
    - $a \land b = b \land a$
  - 結合律
    - $a \lor (b \lor c) = (a \lor b) \lor c$
    - $a \land (b \land c) = (a \land b) \land c$
  - 吸収率
    - $a \lor (a \land b) = a$
    - $a \land (a \lor b) = a$
- また上から以下を証明できる
  - 冪等律
    - $a \lor a = a$
    - $a \land a = a$
- 半順序集合やブール代数の概念も含んだ概念になる
