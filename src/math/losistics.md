# 論理

- $A \land B$
  - AかつB
  - Aが成り立ち、またBも成り立つ
  - and
  - |$A$|$B$|$A \land B$|
    |:-:|:-:|:-:|
    | T | T | T |
    | T | F | F |
    | F | T | F |
    | F | F | F |
- $A \lor B$
  - AまたはB
  - Aのみ成り立つ、Bのみ成り立つ、AとBの両方が成り立つ、のどれか
  - or
  - |$A$|$B$|$A \lor B$|
    |:-:|:-:|:-:|
    | T | T | T |
    | T | F | T |
    | F | T | T |
    | F | F | F |
- $\lnot A$
  - Aでない
  - Aが成り立たない
  - not
  - |$A$|$A \lnot$|
    |:-:|:-:|
    | T | F |
    | F | T |
- $A \implies B$
  - AならばB
  - Aが成り立つときはBが成り立つ
  - Aが成り立たないときは、Bは成り立つか成り立たないかわからない
    - これ自然に説明したい
    - Aが成り立たないときはBに言及していないので、Aが成り立たないときにはBが成り立とうが成り立たないが、全体として成り立つ
    - 例えば $1 = 0$ ならば $100 = 101$ は正しい
    - 覚えたほうが早い
  - |$A$|$B$|$A \implies B$|
    |:-:|:-:| :-: |
    | T | T |  T  |
    | T | F |  F  |
    | F | T |**T**|
    | F | F |**T**|
