# 確率密度

- 無限にデータを増やして、ヒストグラムの幅をどんどん小さくしていくと、なめらかになった(連続な)関数が現れる
- 全体を積分すると1となる
  - 例えば体重のデータなら、$\displaystyle \int_0^\infty P(X=x) = 1$
- 確率密度の関係から確率密度を求める例
  - $1 \leq X \leq 2$ で、$f(x) = ax^{-3}$ であることが分かるとき
  - 1から2まで $f(x)$ を積分してみると、$\dfrac{3a}{8}$ となる
  - $\dfrac{3a}{8} = 1$ より、$a = \dfrac{8}{3}$ とわかる
