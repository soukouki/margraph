# 二項分布

- ベルヌーイ分布の確率分布を使う
- $B_i(n,p)$ で表される
  - $n$ は試行回数
  - $p$ はベルヌーイ分布の確率
- 母数が決まれば分布の形が決まる
- 例えば、試行回数1000、$p=1$ なら、
  - 0から999まで0, 1000のところだけ1になる
- 例えば、試行回数1000、$p=0.5$ なら、
  - 0から1000にかけて、釣鐘状の分布になる
- $B_i(n,p) = P(X=x) = \dbinom{n}{x}\ p^x\ (1-p)^{n-x}$
- 期待値 $E[x] = np$
- 分散 $V[X] = np(1-p)$
